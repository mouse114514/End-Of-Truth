# Contributing

Thanks for your interest in End Of Truth! This is a fan project built with Godot 4.3.

## Getting Started

1. Fork the repository
2. Clone your fork
3. Open `project.godot` in Godot 4.3
4. Create a branch for your changes

## Code Style

- **GDScript**: Follow the [official GDScript style guide](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/gdscript_styleguide.html)
- **No comments unless necessary**: Code should be self-documenting
- **Use English** for variable names, function names, and comments
- **One file per class** (with rare exceptions)

## Pull Request Process

1. Keep PRs focused on a single change
2. Update the README if your change affects user-facing features
3. Make sure the project opens without errors in Godot 4.3
4. Link any related issues in the PR description

## Asset Contributions

- **Art**: Include source files (.psd, .kra, .svg) if possible
- **Music**: Provide as .ogg (compressed) and include the original project file if possible
- **Sound Effects**: Provide as .wav (source) — they'll be compressed for the build

## Code of Conduct

Be respectful. This is a volunteer passion project.
